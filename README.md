## 📋 О проекте

**SecurePass Generator** - это десктопное приложение с графическим интерфейсом для генерации криптостойких паролей и анализа их надежности.

Проект разработан в рамках выполнения практического задания по дисциплине **ОП.03 Основы алгоритмизации и программирование**.

---

## 🎯 Какую проблему решает проект

В современном цифровом мире безопасность личных данных является критически важной. Однако многие пользователи продолжают использовать простые и легко взламываемые пароли:

- **123456** - самый популярный пароль в мире (используется миллионами людей)
- **password** - второй по популярности
- **qwerty** - используется на каждом 1000-м аккаунте
- Повторение одного пароля на разных сайтах
- Использование личной информации (дата рождения, имя)

### Последствия использования слабых паролей:
- 🔓 Взлом аккаунтов за секунды методом перебора
- 💰 Потеря доступа к банковским счетам
- 📧 Компрометация личной переписки
- 🎭 Кража личности
- 💼 Утечка корпоративных данных

### Решение:
✅ Генерация сложных паролей по криптографическим стандартам  
✅ Мгновенная оценка надежности пароля  
✅ Обучение пользователей основам парольной безопасности  
✅ Полное логирование для анализа действий  
✅ Простой и понятный интерфейс

## ⚙️ Функционал приложения

### 🔧 Основные возможности:

### 1️⃣ Генератор паролей

**Настройки генерации:**
- 📏 **Длина пароля:** от 8 до 32 символов (регулируется ползунком)
- 🔤 **Строчные буквы (a-z):** включены по умолчанию
- 🔠 **Заглавные буквы (A-Z):** опционально
- 🔢 **Цифры (0-9):** опционально
- 🔣 **Специальные символы (!@#$%^&*()_+-=[]{}|;:,.<>?):** опционально

**Возможности:**
- ⚡ Мгновенная генерация пароля в одно нажатие
- 📊 Автоматическая оценка надежности сгенерированного пароля
- 📋 Копирование пароля в буфер обмена одной кнопкой
- 💾 Автоматическое сохранение в историю
- 🎨 Цветовая индикация надежности (красный/желтый/зеленый)

**Пример использования:**
Параметры: Длина = 16, Все опции включены
Результат: K#9mP@xL2$vN8qR!
Оценка: Сильный пароль ✅
Unknown

---

### 2️⃣ Анализатор надежности паролей

**Критерии оценки:**
- ✔️ Минимальная длина (8+ символов)
- ✔️ Наличие строчных букв
- ✔️ Наличие заглавных букв
- ✔️ Наличие цифр
- ✔️ Наличие специальных символов
- ✔️ Общая сложность (балльная система)

**Уровни надежности:**
- 🔴 **Слабый** (0-3 балла): пароль легко взломать
- 🟡 **Средний** (4-6 баллов): приемлемая защита
- 🟢 **Сильный** (7+ баллов): высокая защита

**Рекомендации:**
Система предоставляет персонализированные советы:
- "Увеличьте длину до 12+ символов"
- "Добавьте заглавные буквы для усиления"
- "Используйте специальные символы"
- "Добавьте цифры в пароль"

**Пример анализа:**
Введенный пароль: hello123
❌ Оценка: Слабый
⚠️ Проблемы:

    Слишком короткий (8 символов вместо 12+)
    Нет заглавных букв
    Нет специальных символов
    Слово из словаря (hello)
    
---

### 3️⃣ Система логирования

**Что записывается:**
- 🕐 Временная метка каждого действия (дата и время с точностью до секунды)
- 📝 Тип операции (генерация, анализ, просмотр истории)
- 📊 Параметры операции (длина пароля, используемые символы)
- ✅ Результат операции (уровень надежности)
- ⚠️ Ошибки и предупреждения

**Формат записи:**
[2024-05-15 14:30:25] - Приложение запущено
[2024-05-15 14:30:45] - Сгенерирован пароль: длина=16, надежность=Сильный
[2024-05-15 14:31:10] - Проверен пароль: надежность=Слабый
[2024-05-15 14:31:30] - Пароль скопирован в буфер обмена
[2024-05-15 14:32:00] - Открыта история операций
[2024-05-15 14:35:00] - Приложение закрыто

**Файлы логов:**
- `logs/activity.log` - основной журнал действий
- `logs/password_history.txt` - история паролей (без самих паролей для безопасности)

---

- 📜 Просмотр всех выполненных действий в хронологическом порядке

- 📊 Статистика использования приложения
